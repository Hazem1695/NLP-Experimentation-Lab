# NLP Engineering Lab

A comprehensive collection of Natural Language Processing (NLP) projects focused on experimentation, model comparison, feature engineering, and reproducible machine learning workflows.

The goal of this repository is not only to build NLP models, but also to document the complete engineering process behind them:

- Data preprocessing decisions
- Feature extraction techniques
- Model selection
- Hyperparameter experimentation
- Evaluation metrics
- Lessons learned from each experiment

---

## Repository Goals

This repository serves as:

- An NLP learning journey
- A machine learning experimentation lab
- A reproducible benchmark collection
- A portfolio demonstrating practical NLP engineering skills

Every project includes detailed documentation of experiments and observations.
